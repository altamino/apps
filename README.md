## Requirements

Apple devices:
- [if using TrollStore](https://github.com/opa334/TrollStore): iOS 14.0 - 16.6.1, 16.7 RC (20H18), 17.0.
- if using sideloading: iOS 14.0 and higher.

Android:
- 6.0+, armeabi-v7a support (check this using Aida64 for Android).
